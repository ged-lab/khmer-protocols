=============================
The Variant Calling Protocol
=============================

:authors: Aditi Gupta, C. Titus Brown.

This is a protocol for identifying Single Nucleotide Polymorphisms (SNPs) in next-generation sequencing datasets.  It is part of khmer-protocols; see
`the main page for this version <../index.html>`__ for citation
information, and `the khmer-protocols site
<http://khmer-protocols.readthedocs.org/>`__ for the latest released
version.
